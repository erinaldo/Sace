Task: Aguardar Solução da Equipe de Sistemas
Aguarda solução da Equipe de Sistemas para a demanda do usuário.
Purpose
Aguarda solução da demanda enviada pelo usuário da Equipe de Desenvolvimento de Sistemas
Main Description

A Equipe de Atendimento ao Usuário deve aguardar atualizações realizadas no ticket criado no Redmine. As atualizações são recebidas por e-amail quando esta se cadastra como "observadora" do ticket.

As atualizações devem ser utilizadas como base para atualizar o status da demanda do usuário no SOS Manutenção. Assim esse ficará sempre informado do andamento da sua solicitação.

Uma das principais atualizações que deve ser atualizada no SOS Manutenção é a data prevista para atendimento da demanda. Essa data deve ser informada pelo Scrum Master do Produto através do ticket criado no Redmine quando este inicia o atendimento do mesmo.
