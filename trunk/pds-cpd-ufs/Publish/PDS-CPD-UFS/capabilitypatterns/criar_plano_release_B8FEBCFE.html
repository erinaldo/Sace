Task: Criar Plano das Releases
Plano inicial do projeto para a release que será desenvolvida.
Purpose

Mostrar a expectativas de prazos e funcionalidades e suas dependências.

Main Description

No início do projeto o Scrum Master do Produto deverá criar um plano de alto nível para a release que será desenvolvida. Nesse plano deve estar descritos:

  • O número e a duração dos Sprints (o padrão do CPD são 3 semanas)
  • O que deverá ser entregue em cada Sprint (apenas uma idéia inicial que será depois melhor discutida)
  • Pessoas que serão envolvidas no projeto.
  • Número de releases

Para criação do plano de release deve-se utilizar a ferramenta OpenProj.
