Task: Gerenciar Recursos Necessários
Verifica se todos os recursos necessários para o desenvolvimento estão disponíveis.
Purpose

Remover todos os impedimentos para que o projeto seja iniciado.

Main Description

O Scrum Master do Produto deverá listar todos os recursos necessário ao desenvolvimento e verificar se eles estão disponíveis. O Scrum Master pode ser envolvido para ajudar a mitigar os riscos envolvidos.

Devem ser avaliados, por exemplo:

  • Existe hardware necessário para o desenvolvimento da aplicação?
  • Os software estão instalados nas máquinas dos desenvolvedores?
  • A equipe é suficiente? Serão necessários novas pessoas?
  • Se o prazo para conclusão das solicitações é um dos requisitos, então ele é suficiente?
  • etc.
