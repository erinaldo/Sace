Task: Aprovar Projeto
Aprova o projeto de acordo com escopo e prazo inicial definido.
Purpose
Aprovar projeto para execução pela Equipe de Desenvolvimento de Sistemas.
Main Description

Aprovar projeto para execução pela Equipe de Desenvolvimento de Sistemas. O projeto é descrito pelo documento de Visão do Sistema. Cada nova versão do sistema possui um documento de visão atualizado. Quando novas funcionalidades forem adicionadas em um sistema existente, elas deverão ficar em destaque no documento de visão do sistema.

A aprovação desse documento é condição necessária para que um novo sistema ou alterações em um sistema existente possam ser desenvolvidas.

O documento deve ser armazenado e aprovado utilizando o Nuxeo.

Depois da aprovação deve-se criar o sistema (caso não exista) e a release que será desenvolvida no Redmine.
