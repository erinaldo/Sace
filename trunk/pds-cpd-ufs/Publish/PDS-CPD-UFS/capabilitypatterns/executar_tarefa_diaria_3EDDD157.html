Task: Executar Tarefa Diária
O Desenvolvedor executa a tarefa diária seguindo os padrões estabelecidos.
Purpose
Concluir uma tarefa diária com qualidade.
Main Description

O primeiro passo para que o Desenvolvedor inicie a tarefa é mudar a situação da tarefa ao qual foi designado no Redmine.

Quando a tarefa necessitar criar ou alterar algum modelo de dados, deve-se enviar o novo modelo para o administrador de dados para que este possa aprová-lo. É recomendado também pedir a opinião e a aprovação de outros integrantes da Equipe de Desenvolvimento de Sistemas antes de enviar o modelo para aprovação.

Para executar a tarefa deve-se seguir todos os padrões de desenvolvimento disponibilizados no Site de Desenvolvimento do CPD.
