Task: Resolver Impedimentos
Resolver impedimentos que impedem a conclusão de uma determianda tarefa.
Purpose
Remover impedimentos para conclusão de uma tarefa.
Main Description

O Scrum Master do Produto deve resolver todos os impedimentos para conclusão de uma determinada tarefa. Caso necessário o Scrum Master pode ser envolvido para ajudar a remover esses impedimentos.

Quando uma tarefa possui algum impedimento para que possa ser concluída, o Desenvolvedor deve mudar a situação da tarefa no Redmine e documentar esse impedimento na ferramenta. Em seguida o Desenvolvedor deve escolher uma nova tarefa, com a aprovação do gerente, para execução.

Quando o impedimento for removido a tarefa deve ser encaminhada para o Desenvolvedor que detectou o impedimento. A tarefa só voltará a ser executada quando a tarefa executada pelo desenvolvedor no momento for totalmente concluída.
