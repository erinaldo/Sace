Role: Equipe Scrum
A Equipe Scrum constrói o produto que o cliente irá utilizar.
Role Sets: Equipe de Desenvolvimento de Sistemas, Equipe Scrum
Relationships
Main Description

A Equipe Scrum:

  • Seleciona o objetivo do Sprint (Sprint Goal) e especifica os produtos de trabalho necessários;
  • Tem a liberdade de fazer qualquer coisa dentro das diretrizes do projeto para alcançar o objetivo do Sprint;
  • Auto organiza-se e organiza suas atividades; e
  • Apresenta os produtos de trabalho ao Dono do Produto.

Membros do Time Scrum não têm um papel fixo (Ex: programador, designer, testador ou arquiteto). Todos no projeto trabalham juntos para finalizar a lista de atividades que eles coletivamente se comprometeram a realizar durante um Sprint.

Staffing
Assignment Approaches

Membros devem se dedicar em tempo integral, mas pode haver exceções (ex: administrador de banco de dados).

Deve-se alterar membros somente no intervalo entre Sprints.
