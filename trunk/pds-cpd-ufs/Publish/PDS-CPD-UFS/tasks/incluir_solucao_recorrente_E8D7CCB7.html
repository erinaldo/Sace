Task: Incluir Solução Recorrente
Inclui soluções recorrentes no sistema para que este possa enviá-las automaticamente para o usuário.
Purpose
Incluir soluções recorrentes de demandas de usuários para diminuir o trabalho da Equipe de Atendimento ao Usuário.
Relationships
RolesPrimary Performer:
  • Equipe de Atendimento ao Usuário
Additional Performers:
Main Description

A Equipe de Atendimento ao Usuário deve ficar atenta para demandas recorrentes e que possui uma solução padrão. Um exemplo seria como criar a senha para acessar o sistema X?

A demanda e a solução padrão deve ser cadastradas no SOS Manutenção para que este possa informar automaticamente essa solução aos Role: Usuários dos Sistemas no momento do cadastramento da demanda.
