Task: Receber Solução da Equipe de Sistemas
Recebe a solução enviada pela Equipe de Sistemas.
Purpose
Recebee a solução enviada pela Equipe de Sistemas e enviá-la para o usuário.
Relationships
RolesPrimary Performer:
  • Equipe de Atendimento ao Usuário
Additional Performers:
Main Description

Quando o chamado é atendido pela Equipe de Desenvolvimento de Sistemas, a Equipe de Atendimento ao Usuário é avisada pelo Redmine. Esta então deve avisar, através do  SOS Manutenção que o chamado foi resolvido.
