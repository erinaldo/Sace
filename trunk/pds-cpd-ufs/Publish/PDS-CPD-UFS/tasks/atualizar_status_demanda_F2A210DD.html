Task: Atualizar o Status da Demanda
Atualiza o status da demanda do usuário sempre que movas informações são enviadas pela Equipe de Desenvolvimento de Sistemas.
Purpose
Atualiza o status da solicitação para o Usuário, evitando que este precise ligar para o CPD.
Relationships
RolesPrimary Performer:
  • Equipe de Atendimento ao Usuário
Additional Performers:
Main Description

A Equipe de Atendimento ao Usuário deve atualizar o status da demanda através do SOS Manutenção.

Essas atualizações podem ser realizadas sempre que uma solicitação (ticket) criado no Redmine é atualizado pela Equipe de Desenvolvimento de Sistemas.

As atualizações são enviadas automaticamente por e-mail pelo Redmine para Equipe de Atendimento ao Usuário, sempre que a Equipe de Desenvolvimento de Sistemas faz alguma atualização no ticket.

Uma das atualizações mais importantes que a Equipe de Desenvolvimento de Sistemas deve realizar no ticket criado no Redmine é a data prevista para atendimento da solicitação. Essa data deve então ser atualizada pela Equipe de Atendimento ao Usuário no SOS Manutenção.
