Task: Avaliar Solicitação
Avalia a solicitação do usuário para verificar se é uma demanda para Equipe de Sistemas.
Purpose
Fazer a triagem dos problemas/demandas para Equipe de Sistemas.
Relationships
RolesPrimary Performer:
  • Equipe de Atendimento ao Usuário
Additional Performers:
Main Description

A Equipe de Atendimento ao Usuário deve avaliar todas as solicitações enviadas pelos Usuários dos Sistemas.

A partir dessa avaliação a equipe pode tomar duas ações:

  1. Atender o chamado caso possua solução.
  2. Encaminhar o chamado para Equipe de Sistemas utilizando o Redmine.
