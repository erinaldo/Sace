Role: Equipe de Atendimento ao Usuário
Recebe todas as solicitações de usuário e as encaminha, caso necessário, para a Equipe Scrum.
Relationships
Main Description

A Equipe de Atendimento tem as seguintes responsabilidades:

  • Receber todas as solicitações dos Usuários dos Sistemas e encaminhar, caso necessário, para as Equipe Scrum.
  • Caso existam dúvidas sobre a solicitação deve retornar ao usuário para esclarecer todas as dúvidas.
  • Após o encerramento do chamado deve verificar com o usuário se a solicitação foi atendida adequadamente.
