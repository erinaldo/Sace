Task: Planejar Objetivos do Sprint
Planeja quais serão os objetivos a serem alcançados no Sprint.
Main Description

Através de uma reunião entre a Equipe Scrum e demais interessados devem ser definidos quais são os objetivos a serem concluídos no Sprint.

O primeiro passo é a priorização pelo Responsável pelo Produto das tarefas que deverão ser executadas no Sprint. O Backlog do Produto já possui uma priorização inicial e deve ser utilizado como base para essa tarefa. O resultado dessa atividade é o Backlog do Sprint que deve ser armazenado no Nuxeo. Todas as histórias selecionadas para o Sprint deverão ser aprovadas, através do Nuxeo, pelo Responsável pelo Produto.

Em seguida, a Equipe de Desenvolvimento de Sistemas, deve listar para cada história identificada, quais as tarefas que são necessárias para que ela seja concluída. Cada tarefa deve ser executada em 1 dia. Alguns exemplos de tarefas comuns para uma história.

  • Criar Modelo de Dados
  • Criar Camada de Apresentação.
  • Refatorar camada de Dados
  • Criar Rotinas de Validação dos Dados da Tela.
  • Criar Camada de Lógica de Negócio.
  • Criar Camada de Acesso a Dados.
  • Criar e Executar testes de Automatizados.

Todas as tarefas identificadas para que a história seja concluída devem ser cadastradas no Redmine pelo Scrum Master do Produto, para que depois possam ser associadas aos executores.
