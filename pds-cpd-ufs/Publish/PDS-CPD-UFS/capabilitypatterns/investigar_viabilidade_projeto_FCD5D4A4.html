Task: Investigar Viabilidade do Produto
Investiga se os requisitos do produto podem ser construídos dentro do prazo, custo e padrões de qualidade esperados.
Purpose
Garantir que o produto desejado pode ser construído.
Main Description

Utilizando-se as estimativas que foram anotadas no Backlog do Produto deve-se investigar se o projeto é viável para o prazo que foi estimado.

Nesta tarefa também deve-se verificar se os recursos e prazos estipulados pelo Responsável pelo Produto (isso acontece quando o prazo é um requisito) são sufucientes para que o projeto seja concluído dentro do prazo, custo e padrões de qualidade esperados.

O resultado dessa análise pode ser uma renogociação de prazos, deste projeto ou ainda de outros projetos em andamento pela equipe. Outro resultado possível é que o projeto pode ser colocado em espera até que todos recursos necessários para conclusão do projeto sejam disponibilizados.
