Task: Encaminhar Demanda
Encaminha demandas para Equipe de Sistemas quando não possui solução.
Purpose
Encaminhar para Equipe de Sistemas os chamados sem solução.
Relationships
RolesPrimary Performer:
  • Equipe de Atendimento ao Usuário
Additional Performers:
Main Description

Quando não possui solução para a demanda, a Equipe de Atendimento ao Usuário deve encaminhá-la para Equipe de Sistemas.

Para isso deve ser criado um novo chamado (ticket) dentro do Redmine. O ticket deve ser criado da seguinte forma:

  1. O título do ticket deve iniciar com o número do SOS.
    Exemplo: SOS00123-Demanda do Usuário
  2. O conteúdo do ticket deve conter uma explicação detalhada da demanda. Caso a solicitação do usuário através do SOS Manutenção não seja clara, a Equipe de Atendimento ao Usuário deve esclarecer as dúvidas com o solicitante antes de encaminhá-la para Equipe de Sistemas.
  3. A Equipe de Atendimento ao Usuário deve se cadastrar como "observadora" do ticket criado no Redmine. O objetivo é que possa receber por e-mail quaisquer atualizações realizadas no ticket. Essas atualizações devem ser utilizadas par atualizar o status da demanda no SOS Manutenção.

O ticket deve ser encaminhado primariamente para o Scrum Master. Ele é o responsável por determinar quem irá resolver a demanda.

Porém, caso a Equipe de Atendimento ao Usuário, tenha conhecimento num determinado momento, quem é o Scrum Master do Produto, a demanda pode ser encaminhada diretamente para este.
