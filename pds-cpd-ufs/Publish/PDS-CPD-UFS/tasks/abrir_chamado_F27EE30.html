Task: Abrir Chamado
Usuário abre um chamado solicitando a solução de um problema pela Equipe de Sistemas.
Relationships
RolesPrimary Performer:
  • Usuários dos Sistemas
Additional Performers:
Main Description

Sempre que os Usuários dos Sistemas tiverem qualquer problema/demanda relativo a sistemas deve necessariamente abrir um chamado para Equipe de Sistemas.

O chamado deve ser aberto e acompanhado pelos  Usuários dos Sistemas através do SOS Manutenção.
